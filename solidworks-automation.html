Project Overview

At WAM Industries and Maxxit Group, I developed a comprehensive suite of automation tools that revolutionized the CAD-to-manufacturing workflow. These tools eliminated repetitive manual tasks, reduced human error, and accelerated project delivery timelines by automating critical processes in the design-to-production pipeline.

Business Impact

The automation framework reduced drawing export time from 15 minutes per part to under 2 minutes, eliminated dimension extraction errors that previously caused manufacturing delays, and established a standardized workflow adopted across multiple engineering teams. Annual time savings exceeded 500+ engineering hours.

Core Automation Modules

Developed specialized VBA macros targeting different aspects of the manufacturing workflow, from initial design export to final documentation generation.

DXF/DWG Export System

Automated batch export of SolidWorks parts and assemblies to DXF/DWG formats for CAM systems and laser cutting operations.

  • Automatic layer assignment based on geometry type
  • Custom scale and unit conversion
  • Batch processing for multiple files
  • File naming conventions enforcement
  • Export settings standardization

PDF Generation & Drawing Export

One-click PDF generation from SolidWorks drawings with automatic file organization and revision control.

  • Batch PDF export with quality settings
  • Automatic file naming from custom properties
  • Drawing sheet extraction and separation
  • Watermark and stamp insertion
  • Revision tracking integration

Sheet Metal Export Macro

Specialized automation for sheet metal flat pattern exports with bend line detection and manufacturing data extraction.

  • Automatic flat pattern generation
  • Bend line detection and annotation
  • Bend direction analysis
  • K-factor and bend allowance calculation
  • Material thickness validation

Dimension Extraction Tool

Automated extraction of critical dimensions from SolidWorks parts and assemblies for inspection reports and quality control.

  • Sketch dimension extraction
  • Feature dimension parsing
  • Custom property data mining
  • Excel report generation
  • Tolerance analysis automation

Assembly BOM Generator

Automatic bill of materials (BOM) generation with custom formatting for manufacturing and procurement systems.

  • Multi-level BOM creation
  • Part number extraction and validation
  • Quantity rollup calculations
  • Custom property integration
  • ERP system export formatting

Design Validation Scripts

Automated design rule checking to ensure manufacturability and compliance with company standards before release.

  • Material specification validation

  • Drawing view requirement validation

Sheet Metal Export Workflow

Automated 5-step process for sheet metal flat pattern export with manufacturing data extraction.

01
Part Validation

Verifies sheet metal part, validates material properties and thickness specifications.

02
Flat Pattern Generation

Unfolds sheet metal part applying correct K-factor and bend allowance.

03
Bend Line Detection

Extracts bend angles, radii, directions, and geometric relationships.

04
DXF Export with Layers

Exports with intelligent layer assignment compatible with CNC press brake software.

05
Manufacturing Data Report

Generates Excel report with material specs, bend data, and tooling requirements for ERP import.
