About This Project

This personal robotics project explores the practical implementation of Learning from Demonstration (LfD) techniques in robotic manipulation. By combining teleoperation hardware with multi-sensor data analysis, the goal is to understand how robots can learn effective grasping strategies from human operators.

The project builds hands-on experience with sensor fusion, trajectory planning, and data-driven optimization while creating a functional platform that bridges human intuition with robotic execution. This work demonstrates proficiency in ROS development, Arduino integration, and applied machine learning for robotics applications.

Project Objectives

01

Demonstration Collection

Record 15+ human grasping demonstrations via somatosensory glove, capturing joint trajectories and multi-sensor data with success/failure labels.

02

Pattern Identification

Statistically analyze sensor data to identify discriminative features that separate successful grasps from failures.

03

Trajectory Refinement

Apply averaging, filtering, and time-warping to align trajectories with identified success patterns.

04

Validation

Demonstrate ≥25% improvement in grasp success rate compared to raw demonstration replay baseline.

Methodology

  • Data Collection: Record demonstrations via ROS rosbag with synchronized sensor timestamps
  • Feature Extraction: Compute contact timing, acceleration RMS, approach distance, and visibility metrics
  • Statistical Analysis: Compare feature distributions between success/failure groups to find discriminative thresholds
  • Refinement Pipeline: Temporal alignment → trajectory averaging → Butterworth low-pass filtering → timing adjustment
  • Pre-execution Validation: Sensor-based checks ensure conditions match successful demonstration patterns before execution

Project Goal

Demonstrate that multi-sensor analysis can systematically improve robot learning from human demonstrations. This project aims to develop a practical framework for using multiple sensor inputs to refine robotic grasping trajectories and achieve measurable performance improvements in autonomous manipulation tasks.
