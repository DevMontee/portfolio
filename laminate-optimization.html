Project Overview

This project focuses on optimizing composite laminate stacking sequences for aircraft fuselage skin panels. Using classical lamination theory and the Tsai-Wu failure criterion, the optimization balances structural requirements (strength, stiffness) with weight minimization across multiple fuselage bays with varying load conditions.

Technical Approach

The optimization employs a Monte Carlo-based search algorithm to find balanced, symmetric layup configurations that satisfy failure criteria while minimizing mass. Each bay has unique load conditions (Nx, Ny, Nxy) and ply count constraints, with a ply-drop rule enforced to ensure manufacturability between adjacent bays.

Python Implementation

The optimization algorithm combines classical lamination theory with the Tsai-Wu failure criterion. The code generates balanced symmetric layups and evaluates them against failure indices to ensure structural integrity.

import numpy as np

# Material properties for IM7/8552
E1 = 181e3
E2 = 10.3e3
G12 = 7.17e3
v12 = 0.28

XT = 1500
XC = 1200
YT = 40
YC = 246
S = 68
rho = 1.58e-3  # g/mm³ = 1.58 g/cm³ → 1.58e-3 g/mm³

t_ply = 0.125  # mm
area = 1000 * 1000  # mm² (1 m² skin panel assumed per bay for simplicity)

# Standard angles for balanced layups
angles = [0, 45, -45, 90]

# Load cases and ply limits for 4 bays (Nx, Ny, Nxy in N/mm), max plies
bays = [
    {"loads": [50, 40, 13], "max_plies": 20},
    {"loads": [45, 35, 11], "max_plies": 18},
    {"loads": [40, 30, 10], "max_plies": 16},
    {"loads": [35, 25, 9],  "max_plies": 14},
]

# Precompute stiffness matrix
v21 = v12 * E2 / E1
Q11 = E1 / (1 - v12 * v21)
Q12 = v12 * E2 / (1 - v12 * v21)
Q22 = E2 / (1 - v12 * v21)
Q66 = G12
Q = np.array([[Q11, Q12, 0], [Q12, Q22, 0], [0, 0, Q66]])

def rotate_Q(Q, theta):
    theta = np.radians(theta)
    c, s = np.cos(theta), np.sin(theta)
    T = np.array([[c**2, s**2, 2*c*s],
                  [s**2, c**2, -2*c*s],
                  [-c*s, c*s, c**2 - s**2]])
    T_inv = np.linalg.inv(T)
    return T_inv @ Q @ T_inv.T

def tsai_wu_index(stress):
    s1, s2, t12 = stress
    F1 = 1/XT - 1/XC
    F2 = 1/YT - 1/YC
    F11 = 1/(XT*XC)
    F22 = 1/(YT*YC)
    F66 = 1/(S**2)
    F12 = -0.5 * np.sqrt(F11 * F22)
    return (F1*s1 + F2*s2 + F11*s1**2 + F22*s2**2 + F66*t12**2 + 2*F12*s1*s2)

def generate_balanced_symmetric_layup(n_half):
    half = []
    while len(half) < n_half:
        angle = np.random.choice(angles)
        if half.count(angle) == half.count(-angle):
            half.append(angle)
        elif -angle in half:
            half.append(angle)
    return half + half[::-1]

def calculate_MoS_and_mass(layup, N):
    n = len(layup)
    Q_bars = [rotate_Q(Q, ang) for ang in layup]
    z = np.linspace(-n*t_ply/2, n*t_ply/2, n+1)
    A = sum(Q_bars[i]*(z[i+1]-z[i]) for i in range(n))
    epsilon_0 = np.linalg.inv(A) @ N
    min_mos = float('inf')

    for i in range(n):
        sigma = Q_bars[i] @ epsilon_0
        FI = tsai_wu_index(sigma)
        if FI > 0:
            mos = 1/FI - 1
            if 0 < mos < min_mos:
                min_mos = mos
    mass = n * t_ply * rho * area / 1000  # g → kg
    return min_mos, mass

# Optimization loop per bay
np.random.seed(42)
results = []
prev_plies = None
max_iter = 10000

for i, bay in enumerate(bays):
    best_mos = float('inf')
    best_layup = None
    best_mass = None
    max_plies = bay["max_plies"]
    loads = np.array(bay["loads"])

    for _ in range(max_iter):
        n_half = max_plies // 2
        layup = generate_balanced_symmetric_layup(n_half)

        if prev_plies is not None:
            if abs(len(layup) - prev_plies) > 2:
                continue  # enforce ply drop rule

        mos, mass = calculate_MoS_and_mass(layup, loads)
        if 0 < mos < best_mos:
            best_mos = mos
            best_layup = layup
            best_mass = mass
            prev_plies = len(layup)

    results.append({"bay": i+1, "plies": len(best_layup), "mos": best_mos, 
                    "mass_kg": best_mass, "layup": best_layup})

# Output results
total_mass = 0
print("\n=== OPTIMIZED FUSELAGE SKIN RESULTS ===")
for r in results:
    print(f"Bay {r['bay']} | Plies: {r['plies']} | MoS: {r['mos']:.6f} | Mass: {r['mass_kg']:.2f} kg")
    print(f"  Layup: {r['layup']}")
    total_mass += r["mass_kg"]

print(f"\nTotal Fuselage Skin Mass: {total_mass:.2f} kg")

Key Findings

All bays achieved positive margins of safety, ranging from 0.60% (Bay 4) to 1.64% (Bay 2), indicating structural adequacy while maintaining minimal weight penalty. The ply count decreases progressively from 20 plies (Bay 1) to 14 plies (Bay 4) following the load reduction, with a maximum ply drop of 2 plies between adjacent bays ensuring manufacturability. The predominance of ±45° plies across all configurations reflects the high shear loading (Nxy) in the fuselage skin.

Methodology Highlights

Classical Lamination Theory (CLT) was used to compute the ABD stiffness matrices, relating mid-plane strains and curvatures to resultant forces and moments. The Tsai-Wu failure criterion provides a quadratic interaction equation accounting for combined loading states, superior to maximum stress criteria for composite materials. Balanced symmetric layups ensure zero bending-extension coupling (B=0) and thermal stability, critical for aerospace applications.
